# Portfolio Optimization Tool - Frontend

This is the React frontend for the Portfolio Optimization Tool.

## Tech Stack
- **React** (Vite)
- **Tailwind CSS** (v4)
- **Plotly.js** (Visualization)
- **Axios** (API Communication)

## Setup
Refer to the [Main README](../README.md) for full setup instructions.

```bash
npm install
npm run dev
```
